Solution 2: Delete Extra HTML Tags

The original report, #124, on the problem of including a full html file—one with a set of html tags placed within a separate set of html tags—notes that includeHTML() is intended to be used with a file that contains an HTML fragment. To explore this, the file html_file3.Rmd in the www folder was knit to HTML and saved in the www folder as html_file3.html using the option output: html_document; that is, the same process used to create html_file1.html. This document was then opened in the editor and the !DOCTYPE tag and the set of opening and closing html tags were deleted and the document saved. As seen below, both the mathjax mathematical expression and the mhchem chemical formula are displayed correctly. No additional warning appears in the console.
